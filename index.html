3D Surface Deformation Visualization

This page presents an interactive visualization of deformation between two 3D surface meshes. It supports supplementary material for a scientific submission. No personal or institutional information is included.

Description

The viewer shows color-mapped deformation magnitudes and directional displacement vectors between a reference and deformed surface mesh. Red vectors indicate expansion along the surface normal; blue indicates inward deformation.

Notes

  • The underlying code uses vedo, numpy, and scipy.
  • Meshes are anonymized synthetic samples for visualization purposes.
  • Source code and data are intentionally omitted for blind review.
